# MICRO_ECC IS A SUPER COOL PROJECT!
if you ever need eliptic curve stuff I would recommend micro ECC. works greats and super simple + the docs in the .h file!
# INSTALL MICRO_ECC
to make this project you need to install micro-ecc into this dir. you can do this by running the sh scipt provided `sh install_micro_ECC.sh`
